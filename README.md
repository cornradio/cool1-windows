# Cool1 Windows
我的另一个程序：https://github.com/cornradio/cool1 的 windwos版本。
使用dotnet开发。

![image](https://img.shields.io/badge/dotnet-7.0-blue.svg)
![image](https://img.shields.io/badge/Windows-10%2B-blue.svg)
![image](https://img.shields.io/badge/License-MIT-green.svg)
![image](https://img.shields.io/badge/Version-1.0.0-blue.svg)
## 功能
- 程序快速启动和杀掉
- 程序管理和收藏
- 打开程序所在目录、在终端运行（时候命令行程序）
- 其实不只是程序，也可也放文件夹、文档、图片等。这就是个临时窗口你想放啥他都支持！
